Cluster Description
The K-Means algorithm grouped customers into 5 distinct clusters based on their Income and Spending Score.

- Cluster 1: Low Income, Low Score
- Cluster 2: Medium Income, Medium Score
- Cluster 3: Low Income, High Score
- Cluster 4: High Income, High Score
- Cluster 5: High Income, Low Score

Black star markers represent the centroids of each cluster.
